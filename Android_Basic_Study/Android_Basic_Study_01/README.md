1주차 과제
----


1. 숫자와 연산자를 다 입력받은뒤 숫자는 operands 배열에 저장, 연산자는 operators 배열에 저장.
2. 중위 연산식을 사용하여 계산한 후 Result에 결과 출력, 소수점도 표현가능하게 Double형 처리
3. 예외 처리 -> 0으로 나눌 때, 숫자만 입력되었을 때, 처음부터 연산자가 입력되었을 때
